Blog Entry #3

9/16/2021

I came across an example of some text wrapped around a circle and thought I'd see if I could replicate it.

I was brought to this site, which seems to be a compilation of interesting user-submitted CSS techniques. I may return for future blog post inspiration. Anyways, I was met with this snippet when I opened the article: 

        
    <svg viewBox="0 0 500 500">
        <path id="curve" d="M73.2,148.6c4-6.1,65.5-96.8,178.6-95.6c111.3,1.2,170.8,90.3,175.1,97" />
        <text width="500">
            <textPath alignment-baseline="top" xlink:href="#curve">
                Dangerous Curves Ahead
            </textPath>
        </text>
    </svg>

I was a bit taken aback, it looked pretty daunting. My eyes were drawn to the seemingly nonsense string of numbers and characters. But, I was determined to figure it out. I started with the first tag. The article mentions SVG though it never explained what it was. After googling I found it stands for Scalable Vector Graphics and can be used to make 2d graphics within HTML. That was the key to shaping the text. Basically, you draw a shape in SVG, hide that shape, then use textPath to curve text around the invisible shape. The string of characters I noticed were instructions on how the curve is drawn. This one is pretty specific so I'm guessing the curve was drawn out by hand then converted to the path format. It ends up looking like a sine wave.

The only step from there was to make a circular path to wrap my text around. I took one from Stack Overflow, modified the above code replacing the path, and I had my circle text!
