SVG: Scalable Vector Graphics

This week I decided to get a better handle on SVG. I briefly encountered it during last week’s blog entry but I didn’t get that deep into it. We also imported some SVG icons in one of the recent labs so it seemed appropriate. I referenced this tutorial, which explains what SVG is and how to use it:

SVG is a vector graphic format. It differes from raster graphics, like JPEG or PNG. In raster graphics, images are made up of pixels. A 200px by 200px image looks fine as a thumbnail, but when blown up to full screen on a high resolution monitor, it will look pretty blurry. Vector graphics are made up of equations of lines and curves, and thus can be scaled to any size losslessly.

What's cool about SVG is that you can simply write out these equations yourself, you don't need to go draw it up in an illustrator and download it (although that's an option too). After watching through the video above for the crash course, I decided to see what I could come up with using my limited art talents. I ended up with the snowman above. I think it looks pretty cool. Plus, you can zoom in as far as you want to on it and you'll see almost no aliasing at all. A raster graphic probably wouldn't hold up to that. 

      
<svg width = "250" height = "600" id="example" class="example">
  <circle class = "body" cx = 125 cy = 375 r = 120 fill = "white" stroke = "black" stroke-width= "5px"/>
  <circle class = "middle" cx = 125 cy = 260 r = 90 fill = "white" stroke = "black" stroke-width= "5px"/>
  <circle class = "head" cx = 125 cy = 160 r = 60 fill = "white" stroke = "black" stroke-width= "5px"/>
  <circle class = "left-eye" cx = 105 cy = 150 r = 5 fill = "black" stroke = "black" stroke-width= "5px"/>
  <circle class = "right-eye" cx = 145 cy = 150 r = 5 fill = "black" stroke = "black" stroke-width= "5px"/>
  <polygon fill = "orange" points = "135,160 115,160 125,200" stroke = "black"/>
</svg>
